XML Diagram Viewer and Editor
in Java

Authors:
    xmikul69
    xmechl01

Features:
    Class diagram
    Sequence diagram
    MVC pattern
    Saving and loading diagrams in JSON format

How to compile:
    to compile the app and run it:
        mvn javafx:run
    to make JAR:
        mvn make
    to generate JavaDoc:
        mvn javadoc:javadoc
